Ladrillo: (a.k.a Brick)
========

Is the brick or the rectangle in a board or a tilesPicture.
    
    #Creating Ladrillo
    l = Ladrillo(x,y,20,"red") # x      - x coordinate

                               # 20     - anchor (=> x+20 , y+20)
                               # "red"  - Color
